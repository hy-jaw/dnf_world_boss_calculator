# 世界领主[分数-血量]计算器

这是一个用于计算 DNF 世界领主模式分数与血量关系的工具，支持普通模式和挑战模式，能够帮助玩家预估不同分数间领主血量的差距。

## 功能特点

- 支持两种游戏模式：普通模式和挑战模式
- 输入当前分数和目标分数，计算对应的血量值
- 自动计算血量提升百分比
- 可视化展示分数与血量的关系曲线
- 直观标记当前分数和目标分数在曲线上的位置

## 界面预览

程序界面包含以下主要部分：
- 模式选择区域：可切换普通模式和挑战模式
- 参数输入区域：用于输入当前分数和目标分数
- 计算结果区域：显示当前和目标分数对应的血量及提升百分比
- 数据可视化区域：展示分数与血量的关系曲线图
- 使用说明区域：提供简单的操作指南

## 使用方法

1. 选择游戏模式（普通模式或挑战模式）
2. 在输入框中分别填写当前分数和目标分数（需为有效范围内的整数）
3. 点击"计算提升百分比"按钮
4. 查看计算结果，包括当前血量、目标血量及提升百分比
5. 在图表中可直观看到当前分数和目标分数对应的血量位置（红色点为当前，绿色点为目标）"

## 运行环境

- Python 3.x
- 依赖库：tkinter, matplotlib, numpy

## 安装与运行

1. 确保已安装Python 3.x环境
2. 安装所需依赖库：
```bash
pip install -r requirements.txt
```
3. 运行程序：
```bash
python WorldBossCalculator.py
```

## 注意事项
1. 输入的分数需在所选模式的有效范围内
2. 目标分数必须大于当前分数才能进行有效计算
3. 程序支持中文显示，无需额外配置字体
